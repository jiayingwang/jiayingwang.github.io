Computer Network

 2020.9.11  2020.9.14  135  1 min

In this tutorial, I will share knowledge about computer network.

  1. End system: devices at the edge of the network that provides information or services.

  • Which one is the end system?

    (A) Computer ✅   (B) Router   (C) Switch   (D) Twisted-Pair Copper Wire

  • Where is the end system?

    (A) Network edge ✅   (B) The network core

  1. C/S (client/server) model: a client program running on one end system requests and receives information from a server running on another end system.

  • Which one is C/S model?

    (A) Wechat ✅   (B) BitTorrent   (C) Web

  1. P2P (peer-to-peer) model: interconnected nodes ("peers") share resources amongst each other without the use of a centralized administrative system.

  • Which one is P2P model?

    (A) Wechat   (B) BitTorrent ✅   (C) Web

  1. Connection-Oriented Service: It must establish a dedicated connection between the communicating entities before data communication
